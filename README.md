# Jonathan Requeno - Game Developer Portfolio

A professional portfolio website showcasing game development projects, technical skills, and work experience.

## Stack

- React 19
- Vite
- CSS3

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push to GitHub
2. Connect repository to Vercel
3. Deploy

The site will be built and deployed automatically.
